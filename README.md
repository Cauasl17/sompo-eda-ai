# Sompo AI - Previsão de Risco em Equipamentos Agrícolas

## Descrição do Projeto
Este projeto tem como objetivo analisar dados de sensores e uso de equipamentos agrícolas para identificar padrões de risco de falhas. A solução utiliza análise exploratória de dados (EDA) e conceitos de inteligência artificial para apoiar decisões preventivas.

---

## Objetivo
Prever e entender fatores que influenciam o risco de sinistros em equipamentos agrícolas, auxiliando na redução de falhas e na melhoria da produtividade.

---

## Etapas do Projeto

- Definição do problema (CRISP-DM)
- Criação de dataset simulado coerente com o cenário agrícola
- Limpeza e tratamento dos dados
- Análise exploratória (EDA)
- Identificação de variáveis mais relevantes para risco

---

## Arquivos do Projeto

- notebook.ipynb → análise completa no Colab
- dataset.csv → dados utilizados na análise
- README.md → explicação do projeto

---

## Principais Variáveis Analisadas

- Temperatura do motor
- Vibração da máquina
- Horas de uso
- Idade do equipamento
- Manutenção atrasada

---

## Insights Iniciais

- Equipamentos com maior temperatura apresentam maior risco de falhas
- Vibração elevada está fortemente associada a problemas mecânicos
- Máquinas mais antigas tendem a ter maior probabilidade de sinistro
- Falta de manutenção aumenta significativamente o risco operacional
